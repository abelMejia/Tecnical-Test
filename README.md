## Prerequisites
- **Node.js** `>= 20`
- **NPM** `>= 8` (or **Yarn** / **pnpm**)

## Installation
1. **Clone the repository**
```sh
   git clone https://github.com/abelMejia/Tecnical-Test.git
```

2. **Navigate to the project directory**:
```
  cd Tecnical-Test
```

3. **Install dependencies**:
```
  npm install
```

4. **run the development server:**

```bash
npm run dev
```

## Available Scripts
```bash
npm run build
npm run start
npm run lint
```
